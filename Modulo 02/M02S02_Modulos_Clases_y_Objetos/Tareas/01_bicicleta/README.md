Bicicleta
================================================

Problema
--------

En el archivo `src\exercise.py` implementar una clase llamada `Bicicleta`:

* El constructor de la clase debe recibir como parÃ¡metros el color (str) y la altura en metros (float) de la bicicleta, en ese orden, para almacenarlos en los atributos `color` y `altura`.


Validaciones
------------

Ejemplo 1:
```python
>>> b = Bicicleta('greenyellow', 0.96)
>>> b.color
'greenyellow'
>>> b.altura
0.96
```

Ejemplo 2:
```python
>>> b = Bicicleta('antiquewhite', 1.06) 
>>> b.color
'antiquewhite'
>>> b.altura
1.06
```
